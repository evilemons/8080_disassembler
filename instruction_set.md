Abbreviations:

  * R:   any register (A, B, C, D, E, H, L)
  * u8:  byte
  * u16: word
  * adr: 16 byte address 
  * PC:  program counter
  * SP:  stack pointer

8-Bit Transfers:

| Instruction |             Description                |
| :---------  | :------------------------------------: |
| MOV R R     | Move data from one register to another |
| MVI R u8    | Load u8 into register                  |
| LDA adr     | Load u8 from adr into (A) register     |
| STA adr     | Stores u8 at adr                       |
